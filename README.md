# Portfolio

웹 프론트엔드 개발자 포트폴리오입니다. 🚀



## Build With

- Vue.js
- Nuxt.js
- Scss



## Build Setup

``` bash
# install dependencies
$ npm install # Or yarn install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
